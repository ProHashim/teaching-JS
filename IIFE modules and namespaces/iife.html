Lexical Scope

Lexical scope is the ability to access variables and functions from within a function. " Functions are executed using the scope chain that was in effect when they were defined" - according to JavaScript Definition Guide.

Function expression

Function expression is a function declaration that is assigned to a variable.

Unnamed Function or anonymous function 

            
                var x = function() {
                    return "Hello World";
                }

Unnamed function can be used as a callback function.

IIFE - Immediately Invoked Function Expression

An Immediately Invoked Function Expression (IIFE) is a function that is automatically invoked when it is defined. 

        (function() {
          // code here
        })();
